Contact Form 7 messages merged with Swal Alert.
== Description ==
You need Contact Form 7 installed.
This plugin does not put an entire contact form inside a swal alert but just the response message when you click the Submit Button.
== Installation ==1. Make sure you have Contact Form 7 installed.
2. Download the file to install and activate.
3. Done! Just remember to keep the plugins up to date!
